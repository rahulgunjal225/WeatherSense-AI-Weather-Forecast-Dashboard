
# 🌦️ WeatherSense - AI Weather Forecast Dashboard

A modern AI-powered Weather Forecast Dashboard built with **Python, Streamlit, Plotly, Pandas, and OpenWeatherMap API**.

## 🚀 Live Features

- 🌍 Real-time weather data
- 📅 5-Day Weather Forecast
- 📊 Interactive Plotly Charts
- 🤖 AI Weather Insights
- 🌡️ Temperature Monitoring
- 💧 Humidity Tracking
- 💨 Wind Speed Analysis
- 🌥️ Cloud Coverage
- 👁️ Visibility Information
- 🌅 Sunrise & Sunset Details
- ❤️ Favorite Cities
- 🕒 Search History
- 🌙 Dark Mode Support
- 🔄 Auto Refresh

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rahulgunjal225/WeatherSense-AI-Weather-Forecast-Dashboard.git
cd WeatherSense-AI-Weather-Forecast-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```

---

## 🔑 API Configuration

Create a file:

```text
.streamlit/secrets.toml
```

Add:

```toml
OPENWEATHERMAP_API_KEY = "YOUR_API_KEY"
```

Get your free API key from:

https://openweathermap.org/api
